Splice Presents Sounds of KSHMR Vol 1 - Plugins presets
-------------------------------------------------------

Enjoy the personal collection of synth presets from one of the most respected EDM producers.
KSHMR put together an amazing selection of Xfer Records Serum and Lennard Digital Synlenth1  
presets.                                                                                     
                                                                                             
Note that you need the latest version of the plugins, the presets are NOT compatible with    
* Sylenth1 v2.2.0 or older                                                                   
* Serum v1.0                                                                                 
                                                                                             
Go to https://www.xferrecords.com/ to download the latest version of Serum                   
Go to https://www.lennardigital.com/ to download the latest version of Sylenth1              

How to install the presets:
===========================

Serum:

To install the Serum presets, you need to copy 2 folders to the right
location:

Copy the the folder called `Sounds of KSHMR Vol 1` from
  `KSHMR Sylenth1 Presets` > `Put contained folder in Serum Noises`
to
  On OS X:
    `/Library/Audio/Presets/Xfer Records/Serum Presets/Noises/`
  On Windows:
    `/Documents/Xfer/Serum Presets/Noises`

Copy the the folder called `Sounds of KSHMR Vol 1` from
  `KSHMR Sylenth1 Presets` > `Put contained folder in Serum Presets`
to
  On OS X:
    `/Library/Audio/Presets/Xfer Records/Serum Presets/Presets/`
  On Windows:
    `/Documents/Xfer/Serum Presets/Presets`

Sylenth:

From within the plugin, load the `Sounds Of KSHMR Sylenth1 Soundset.fxb` file
available from the `KSHMR Sylenth1 Presets` folder.
